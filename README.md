# Vue Fes Japan 2025

## Requirement

- Node.js
- pnpm

For version information, please refer to package.json.

## Setup

```sh

# clone via ssh
git clone git@github.com:vuejs-jp/vuefes-2025.git

# install dependencies
pnpm install

# launch application dev server and storybook
pnpm dev
```
